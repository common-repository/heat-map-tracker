== Installation ==

### How to install ###
1. Download the ZIP package.
2. Open WordPress admin and go to Plugins -> Add New -> Upload. Browse for the ZIP file spy-analytics-lite.zip on your computer and hit “Install Now”.
3. Activate the plugin.

### Quick Start Quide ###

1. Install HeatMapTracker Lite
2. Open HeatMapTracker Lite - Settings. Check Enable record checkbox.  Save settings
3. After a while, open HeatMapTracker Lite - Heat Maps section. If you have recorded data you will see it in the list.
4. Choose one of recorded session in the list and click "Generate Heatmap"
5. Enjoy with the results
